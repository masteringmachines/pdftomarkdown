— Process

How it works

01

Parse & detect

PDF.js reads every word with its exact font size and (x, y) position. Pages below the character threshold trigger automatic OCR fallback via pytesseract in the Python CLI.

02

Structure & classify

Words are grouped into lines by vertical proximity. Font size thresholds classify headings. Tables are identified by column alignment and separated from the text flow.

03

Render Markdown

Tables become pipe tables. Headings get # prefixes. Metadata becomes YAML frontmatter. Output is clean, lint-passing Markdown — copy it or download as .md.
